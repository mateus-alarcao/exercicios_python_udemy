# Exercícios de Python - Curso Udemy

Este repositório contém os exercícios que desenvolvi durante meus estudos no curso de Python pela [Udemy](https://www.udemy.com/). O objetivo é praticar a linguagem, reforçar os conceitos aprendidos e acompanhar minha evolução na programação.

# Sobre o Curso

Curso realizado na plataforma Udemy. Ele aborda fundamentos de Python e inclui diversos exercícios práticos que foram resolvidos passo a passo.

# Tópicos abordados:

- Sintaxe básica do Python
- Variáveis e tipos de dados
- Estruturas de decisão (`if`, `else`, `elif`)
- Estruturas de repetição (`for`, `while`)
- Listas, tuplas e dicionários
- Funções e escopo
- Módulos e pacotes
- Tratamento de erros
- Programação orientada a objetos (POO)
- Entre outros...

# Objetivo

- Praticar Python na prática com exercícios variados.
- Reforçar a lógica de programação.
- Ter um histórico dos meus estudos públicos e acessível.
